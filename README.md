# 我全都要

> 太吾绘卷 Mod——集多种便利功能于一身。

## 功能

### 1. 开局特质无限
- 创建角色时特质选择不受点数限制，所有特质均可自由选择
- 显示"无限"点数

### 2. 物品重量归零
支持以下分类独立开关，可在游戏中自由开启/关闭：

| 分类 | 包含类型 |
|------|---------|
| 📚 书籍 | SkillBook |
| 🍖 食物 | Food |
| 💊 药物茶酒 | Medicine, TeaWine |
| ⚔️ 装备 | Weapon, Armor, Accessory, Clothing, Carrier |
| 🔧 工具 | CraftTool |
| 🪨 材料 | Material |
| 📦 杂项 | Misc |
| 🦗 促织 | Cricket |

### 3. 五行锁定混元
- 开启后台吾本人的五行类型强制为混元（均衡分配 20/20/20/20/20）
- 不影响其他 NPC

### 4. 自定义开局点数
支持通过滑条调节以下点数上限：

| 项目 | 范围 |
|------|------|
| 主属性总点数 | 300 ~ 540 |
| 技艺资质总点数 | 800 ~ 1440 |
| 功法资质总点数 | 700 ~ 1260 |
| 特质总点数 | 7 ~ 90 |

### 5. 突破百分百成功 ⚠️ 待测试
- 开启后功法突破时每个格子必定成功
- 支持游戏内热变配

## 安装

将 `Mod/我全都要/` 文件夹放入游戏根目录下的 `Mod/` 目录中，启动游戏并在 Mod 管理界面启用。

```
游戏根目录/
├── Mod/
│   └── 我全都要/
│       ├── Config.lua
│       ├── cover.png
│       └── Plugins/
│           └── GrantAllTraitsMod.dll
├── The Scroll Of Taiwu.exe
└── ...
```

## 设置说明

- 重量开关和五行锁定混元：支持在游戏中直接切换，即时生效
- 开局特质无限和自定义点数：修改后需**重启游戏**生效
- 自定义点数通过滑条在 Mod 管理界面中调节

## 兼容性

- 适用于太吾绘卷正式版
- 前后端均注册为插件，支持游戏 Mod 管理框架
- 与其他不修改特质界面和物品重量的 Mod 兼容
